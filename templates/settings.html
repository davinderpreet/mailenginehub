How to set up Amazon SES:
  1. Create a free AWS account at aws.amazon.com
  2. Go to SES → Verified Identities → Verify your sending email
  3. Create IAM user with AmazonSESFullAccess permission
  4. Copy the Access Key ID and Secret Access Key below
  5. Request to exit "sandbox mode" so you can email any address
