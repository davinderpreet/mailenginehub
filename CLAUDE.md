## Architecture

```
Shopify webhooks -> identity_resolution -> Contact
Website pixel -> CustomerActivity
SES webhooks -> BounceLog / engagement updates

Nightly 2-6am UTC:
  shopify_sync -> ai_engine (RFM) -> customer_intelligence (50+ fields)
  -> next_best_message (10 actions/contact) -> campaign_planner (opportunities)
  -> outcome_tracker -> learning_engine -> strategy_optimizer

Sending: enqueue -> delivery_engine (warmup) -> email_sender (SES) -> inbox
Studio: knowledge + performance -> studio_skills (6 AI skills) -> candidate -> approve -> template
```
